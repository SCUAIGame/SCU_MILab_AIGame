##简介
- AIGame研究小组展示网页，基于bootstarp+flask搭建

##环境搭建
- 基于python2.7,需要安装以下插件
 - pip install flask
　- pip install flask-bootstarp
 - pip install flask-mail
 - pip install flask-wtf

##开始使用
- python run.py启动后台,输入127.0.0.1:5000/index访问主页
